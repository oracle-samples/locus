## What OS / App Server / Application technology?

> Static HTML documentation only. Source content is authored in Markdown and built locally with MkDocs (Python) using the Material theme. The resulting plain HTML, CSS, and JavaScript files are published by GitHub Actions to GitHub Pages, which serves them through GitHub's CDN with HTTPS via Let's Encrypt-issued certificates. There is no operating system instance, no application server, no backend logic, and no database — the deployed artefact is a tree of static files.
